# 构建工具

## Windows

#### 脚本说明

* config.bat：配置信息
* clean.bat：清理脚本
* build.bat：构建脚本：Java运行环境、项目、启动器
* startup.bat：启动脚本

#### 依赖环境

* [Java](http://openjdk.java.net/)：11+
* [Maven](http://maven.apache.org/)
* [CMake](https://cmake.org/)
* [Visual Studio](https://visualstudio.microsoft.com/zh-hans/vs/)

> 脚本只能在builder目录中执行
